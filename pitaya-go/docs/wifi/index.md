# Wi-Fi Tutorials

## Introduction

The Wi-Fi tutorials describe the use of the ATWINC1500B Wi-Fi Network Controller on Pitaya Go to build state-of-the-art Internet of Things (IoT) applications.

These tutorials consist of several example codes and projects. The examples are organized in different levels of codes to explain Wi-Fi Network Controller API usage â€“ from basic Wi-Fi operations to advanced topics. These examples are categorized as:

* Basic Examples

	- [Chip Info Example](basic/chip-info-example.md)
	- [MAC Address Example](basic/mac-address-example.md)
	- [STA Mode Example](basic/mode-sta-example.md)
	- [Signal Strength Example](basic/signal-strength-example.md)
	- [AP Mode Example](basic/mode-ap-example.md)
	- [Power Save Mode Example](basic/power-mode-example.md)
	- [HTTP Provision Example](basic/provision-http-example.md)

* Protocol Examples

	- [Ping Request Example](protocol/ping-example.md)
	- [TCP Client Example](protocol/tcp-client-example.md)
	- [TCP Server Example](protocol/tcp-server-example.md)
	- [UDP Client Example](protocol/udp-client-example.md)
	- [UDP Server Example](protocol/udp-server-example.md)
	- [NTP Time Client Example](protocol/time-client-example.md)

* Advanced Examples

	- [MQTT Client Example](advanced/mqtt-client-example.md)
	- [SSL Connection Example](advanced/ssl-connect-example.md)
	- [Weather Client Example](advanced/weather-client-example.md)
